*opspacing.txt* A plugin that sets correct spacing between operators

Author: Alberto Sadde
License: This plugin is free software

INTRODUCTION                        *opspacing*

This plugin is intended to fix other developer's bad choices when coding.
It will change, for example, (1+2) to (1 + 2), true==false to true == false
making the file more readable.

COMMANDS                        *opspacing-commands*

:CorrSpacing            This command will set the correct spacing for all
                        operators across the whole file

MAPINGS                         *opspacing-mappings*

<leader>e                   Is the default mapping
                            User can add its custom mapping to vimrc
